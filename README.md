# Axel Take Home

This the take home solution built by Germain Boue

## Design Decisions

- Use of server components, everything rendered on the server (reduced bundle size / fast performance)
-

## Installation

Step 1:

```bash
yarn install
```

Step 2:

```bash
yarn dev
```

Step 3:
Open chrome browser and go to http://localhost:3000/
